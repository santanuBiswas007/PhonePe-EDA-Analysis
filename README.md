# PhonePe Transaction Analysis using Python

## Project Overview
Performed Exploratory Data Analysis (EDA) on PhonePe transaction data to uncover user behavior, transaction trends, device preferences, and district-level digital payment adoption.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Identified top states by transaction volume and value.
- Analyzed transaction type contributions.
- Studied smartphone brand dominance across states.
- Measured user penetration using demographic data.
- Examined correlation between population and registered users.
